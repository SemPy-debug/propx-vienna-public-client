# Troubleshooting

## The license check fails

PropX requires internet access for the online license check. The license uses server time.

## The workflow fetch fails

The client downloads live public data from Vienna OGD/GIS, OpenStreetMap, and configured open sources. Check internet access and retry the specific fetch step from the GUI.

## The public trial is expired or the machine limit is reached

Contact `amvrazis.s@gmail.com` for a private key or reset.
